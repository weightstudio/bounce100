本政策說明 （下稱「本服務」）如何蒐集、使用與保護您的個人資料，以及您可如何行使個人資料相關權利。使用本服務即表示您同意依本政策蒐集與處理資料。

我們蒐集哪些資料

  • 您主動提供： 暱稱、顯示名稱；（若有）電子郵件或第三方登入提供之公開資訊。
  • 系統自動產生： 排行榜與遊戲紀錄（分數、排名）、裝置與技術資訊（IP、OS/App 版本）、使用分析、崩潰日誌。

蒐集目的與使用方式

  • 提供與維運服務（顯示排行榜、儲存分數與進度）。
  • 改善遊戲體驗與效能之分析。
  • 安全與防濫用（反作弊、維持排行榜公平）。
  • （選用）提供更新與重要變更之通知。

第三方服務

本服務可能使用平台或第三方（例如 Google Play Games、Apple Game Center、Sign in with Apple、Firebase 等）以提供登入、排行榜、雲端儲存或分析等功能；其資料處理遵循各服務之隱私權政策。

資料保存期間

我們會於達成蒐集目的所需期間內保存資料，並依法律義務、爭議處理或安全需求而適度延長。您得依「資料主體權利」要求刪除或存取。

如何刪除或移除資料（Android／iOS 通用）

  • 平台帳號資料（排行榜、成就、平台登入）： 若您使用 Google Play Games（Android）或 Apple Game Center / Sign in with Apple（iOS），該等資料通常由平台管理。您可以：
    • Android（Google Play Games）： Play 商店 App → 設定 → Play 遊戲設定 → 隱私與設定 → 刪除個人檔案與遊戲資料，在列表中選擇 ；或至 Google 帳號的「第三方存取權」移除本服務之存取。
    • iOS（Game Center / Sign in with Apple）： 設定 → Game Center 管理您的帳號；若使用「使用 Apple 登入」，請至 設定 → Apple ID → 密碼與安全性 → 使用 Apple ID 的 App（或 appleid.apple.com）選取本服務並停止使用。若遊戲資料儲存在 iCloud，請至 設定 → Apple ID → iCloud → 管理儲存空間 找到本服務並刪除資料。
  • 我們自行保存的資料（如有）： 若我們另行保存您的資料（例如聯絡 Email 或備份暱稱），您可透過下方聯絡方式提出刪除請求，我們將協助刪除。

資料分享與揭露

  • 受託之服務供應商（僅限提供服務必要之範圍）。
  • 依法令或政府機關合法請求。
  • 維護合法權益、防止詐欺或保護使用者安全。
  • 經您同意之其他情形。

資料安全與跨境傳輸

我們採取合理之技術與管理措施保護資料；資料可能在不同司法轄區處理與保存，使用本服務即表示您同意此類跨境傳輸。

您的權利

您可請求存取、更正、刪除、或限制處理等權利；我們可能需驗證您的身分後再處理。

政策變更

本政策可能因服務或法規調整而更新；重大變更將於頁面公告並更新「最後更新日期」。

Retention

Data is retained for as long as necessary for the purposes above and for legal, dispute-resolution, or security needs. You may request deletion or access as described below.

How to Delete or Remove Access (Android / iOS)

  • Platform account data (leaderboards, achievements, platform sign-in): If you use Google Play Games (Android) or Apple Game Center / Sign in with Apple (iOS), such data is typically managed by the platform. You can:
    • Android (Google Play Games): Play Store app → Settings → Play Games Settings → Privacy & settings → Delete your Play Games data, then select ; or remove the app’s access from your Google Account’s “Third-party access”.
    • iOS (Game Center / Sign in with Apple): Settings → Game Center; and for “Sign in with Apple”, go to Settings → Apple ID → Password & Security → Apps Using Apple ID (or appleid.apple.com) to stop using Apple ID with this app. If game data is stored in iCloud, go to Settings → Apple ID → iCloud → Manage Storage to delete the app’s data.
  • Data we store ourselves (if any): If we store additional data (e.g., contact email or backup nickname), contact us via the channels below and we will delete it upon request.
